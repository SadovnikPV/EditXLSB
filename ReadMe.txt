EditXLSB 
version 0.1

Описание
Программа находит в каталоге на уровень выше себя, файл с расширением *.xlsb (под любым именем), очищает в нем столбец под заголовком "ФИО" и сохраняет файл под именем Таблица_безфио.xlsx в том же каталоге. Исходный файл с расширением *.xlsb удаляется.


Комментарии
1. Подразумевается, что файл *.xlsb в корневом каталоге один. Если файлов много, работа программы не предсказуема.
